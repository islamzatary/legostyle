Slide Show Proc

1) Slide Show with Blts Paginator



Code:

set img_gallary [list]
lappend img_gallary [byt_images_root]/career_services/header_reasons_join_en.jpg
lappend img_gallary [byt_images_root]/career_services/header_reasons_join_ar.jpg
lappend img_gallary [byt_images_root]/career_services/header_reasons_join_fr.jpg

ns_puts [slide_show -full_path -paginator 1 -unique_id 1 -width "980px" -height "358px" $img_gallary]

2) Slide Show with Number Paginator

1 2 3


Code:

set img_gallary [list]
lappend img_gallary [byt_images_root]/career_services/header_reasons_join_en.jpg
lappend img_gallary [byt_images_root]/career_services/header_reasons_join_ar.jpg
